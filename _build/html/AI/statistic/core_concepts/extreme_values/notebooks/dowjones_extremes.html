Avant-propos

Ce rapport s’inscrit dans le cadre de l’Executive Master Statistique et Big Data de l’Université Paris Dauphine et le cours des Valeurs Extrêmes.
Les méthodes employées sont reprises du cours enseigné par Jonathan EL METHNI aux étudiants du Master 2 de la session 2020/2021.
Cette étude est produite avec le logiciel R.

NB1 : souvent dans ce rapport nous écrirons modèle de Frechet/Weibull pour désigner à la fois le modèle de Frechet ($\gamma>0$) et le modèle de Weibull ($\gamma<0$).
NB2 : un rapport plus détaillé sur le jeu de données Marseille est disponible. Ce second rapport sera plus synthétique.

Introduction

L’indice Dow Jones est le plus vieil indice boursier au monde. C’est l’indice boursier de New York qui regroupe les cotations de 30 sociétés. Son équivalent français est le CAC40.

Le jeu de données dowjones reprend le cours quotidien de l’indice sur 5 ans du 11 septembre 1995 au 7 septembre 2000.
